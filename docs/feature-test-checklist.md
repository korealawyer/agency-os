## 1. 🔐 인증 (Authentication)

### 1-A. 로그인 플로우
| # | 점검 항목 | 조작 방법 | 예상 결과 |
|---|---|---|---|
| 1-01 | 로그인 폼 렌더링 | `/login` 접속 | 이메일·비밀번호 입력창 + 로그인 버튼 |
| 1-02 | 정상 로그인 | `admin@agency.com` / `password123` 입력 → 제출 | `/dashboard` 리다이렉트 |
| 1-03 | 틀린 비밀번호 | 잘못된 비밀번호 입력 → 제출 | 에러 메시지 + 로그인 차단 |
| 1-04 | 5회 실패 잠금 | 5회 연속 실패 | `15초 후 다시 시도하세요` 메시지 |
| 1-05 | 빈 필드 제출 | 비밀번호 비우고 제출 | 브라우저 유효성 검사 또는 에러 메시지 |

### 1-B. 회원가입·약관
| 1-06 | 회원가입 폼 전환 | "무료 체험 시작하기" 클릭 | 이름/대행사명/이메일/비밀번호 폼 |
| 1-07 | 비밀번호 찾기 전환 | "비밀번호를 잊으셨나요?" 클릭 | 이메일 입력 폼 |
| 1-08 | 이용약관 모달 | 약관 링크 클릭 | 모달 팝업 + 내용 + 닫기 |
| 1-09 | 개인정보처리방침 모달 | 개인정보 링크 클릭 | 모달 팝업 + 내용 + 닫기 |

### 1-C. 세션·보안
| 1-10 | JWT 세션 유지 | 로그인 후 새로고침 | 세션 유지됨 (대시보드 유지) |
| 1-11 | 로그아웃 | 사이드바 로그아웃 클릭 | `/login`으로 리다이렉트 |
| 1-12 | 미인증 접근 차단 | 로그아웃 상태에서 `/dashboard` 접속 | `/login`으로 리다이렉트 |
